CPU - Procesador

El cerebro del ordenador. Ejecuta instrucciones y realiza millones de cálculos por segundo.

Núcleos Hilos Frecuencia (GHz) Caché L3

RAM

Memoria volátil ultrarrápida. Cuanta más RAM → más programas abiertos sin lentitud.

Tipos actuales: DDR4 y DDR5

HDD vs SSD

SSD: hasta 10 veces más rápido que HDD
HDD: más barato por GB, ideal para almacenamiento masivo

Modelo OSI (7 capas)

  1. Física (cables, señales)
  2. Enlace (MAC, switches)
  3. Red (IP, routers)
  4. Transporte (TCP/UDP)
  5. Sesión
  6. Presentación
  7. Aplicación (HTTP, DNS)

Dirección IP

Identificador único en la red.

IPv4: 192.168.0.1 → 32 bits
IPv6: 2001:0db8::1 → 128 bits

Router vs Switch

Switch: conecta dispositivos en la MISMA red (LAN)
Router: conecta REDES diferentes (Internet)
